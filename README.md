# 📊 Enterprise Telco Customer Churn Intelligence Dashboard

An enterprise-grade **Power BI Business Intelligence solution** built using the **IBM Telco Customer Churn Dataset (7,043 customers)**. This project combines **SQL-based data preparation**, **Power BI data modeling**, **advanced DAX calculations**, and **interactive dashboards** to identify customer churn drivers, quantify revenue at risk, and support executive-level retention strategies.

---

## 📌 Project Overview

Customer churn is one of the most significant challenges faced by telecommunications companies. This dashboard enables business stakeholders to monitor churn performance, evaluate financial impact, analyze customer behavior, and simulate retention strategies through interactive Power BI reports.

The solution follows an end-to-end analytics workflow from raw data preparation to executive decision-making.

The analytical model was designed for performance and scalability.
